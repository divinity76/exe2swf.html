# exe2swf.html
.exe to .swf extractor running in your browser. Extract .swf file from .exe "projector executable" created with Adobe Flash.

Try it here:  https://rawgit.loltek.net/https://raw.githubusercontent.com/divinity76/exe2swf.html/refs/heads/main/exe2swf.html

Extraction algorithm provided by http://www.nullsecurity.org/article/extracting_swf_from_flash_projector
